# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),  
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2026-06-24

### Added
- Statamic 6 support (now compatible with Statamic 5.x and 6.x).
- Internationalization support with English and German translations.
- Translation loading via service provider.

### Fixed
- Control Panel stats widget now renders correctly on the redesigned Statamic 6 control panel (native panel markup, inline external-link icon).

## [1.1.0] - 2025-06-01

### Added
- New **Team ID** configuration option.

### Chore
- Added tests.

## [1.0.0] - 2025-05-31

### Added
- Initial release.
